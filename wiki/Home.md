## What is Platskript?

**Platskript** (`.plats`) is a fun programming language where you code using Flemish dialect words instead of English keywords. It transpiles to Python, so you get all of Python's power with a distinctly Belgian flavor!

```platskript
# coding: vlaamsplats
plan doe
  zet naam op tekst Claude amen
  klap tekst Dag plakt spatie plakt da naam plakt tekst ! amen
gedaan
```

**Output:** `Dag Claude!`

## Quick Language Reference

### Program Structure

```platskript
# coding: vlaamsplats
plan doe
  <statements>
gedaan
```

### Statements

| Syntax | Description |
|--------|-------------|
| `klap <expr> amen` | Print expression |
| `zet <var> op <expr> amen` | Assign variable |
| `roep <func> met <args> amen` | Call function |
| `geeftterug <expr> amen` | Return value |

### Expressions

| Syntax | Description |
|--------|-------------|
| `tekst hello world` | String literal |
| `getal 42` | Number literal |
| `da <name>` | Variable reference |
| `spatie` | Space character |
| `<expr> plakt <expr>` | String concatenation |

### Operators

| Platskript | Python | Description |
|------------|--------|-------------|
| `derbij` | `+` | Addition |
| `deraf` | `-` | Subtraction |
| `keer` | `*` | Multiplication |
| `gedeeld` | `/` | Division |
